# Static assets

## Your hero portrait

Save your photo here as **`portrait.png`** (exact name, this folder):

    public/portrait.png

It appears automatically in the homepage hero. Until the file exists, the hero
shows a monogram placeholder — nothing breaks.

Tips:
- A **transparent cutout** (background removed) looks best over the design, but a
  normal photo works too — the frame contains it.
- The frame is portrait-shaped (4:5) and crops from the top; a head-and-shoulders
  shot works well.
- Your source photo is already high-resolution, so keep it full-size for a crisp
  (HD) result.

### Free ways to remove the background (keeps HD)
- **Adobe Express** background remover — free, keeps full resolution
- **Canva** — Edit image → Background Remover (free tier keeps good resolution)
- **photoroom.com** or **remove.bg** — fast, portrait-friendly

Export as PNG (transparent), rename to `portrait.png`, drop it in this folder.
